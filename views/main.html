DSL-Comet

A Domain Specific Language COllaborative Modelling EnvironmenT


DSL-Comet is a prototype tool that allows model working on mobile devices.

From this web you can update the needed palettes and ecores. Once this is done the application will be able to download them and show the appropiate editor.

You will be ready to manipulate the created instances and to modify their attributes.

This tool offers too a exporting option, which allows you to save the generated model either on the server or on the local device.
